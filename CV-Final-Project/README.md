# CV-Final-Project Dataset
This folder contains the raw and processed imagery for the 2015–2024 scallop survey.

### Google Drive Link
https://drive.google.com/drive/folders/1tnbiZDiW9gCK2h11rDp5K33cW8ox9y9h?usp=drive_link

### Contents
- `dataset/`: Subfolders for each year (2015, 2016, 2018, 2019, 2022, 2024).
- `index.csv`: Metadata containing Altitude and FOV for Part 2 size analysis.

### Action Required
Add this folder as a shortcut to your **MyDrive** so the paths in the Colab notebook (`/content/drive/MyDrive/CV-Final-Project/`) work correctly.
